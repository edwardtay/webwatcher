About WebWatcher

Overview

WebWatcher is an advanced cybersecurity agent built on VeriSense and AgentKit. It provides comprehensive security analysis, threat detection, and monitoring capabilities for both Web2 and Web3 environments.

Key Features

  • Website Security Scanning - Detect phishing red flags and security risks using A2A (Agent-to-Agent) coordination
  • CVE Vulnerability Search - Search Common Vulnerabilities and Exposures database
  • Blockchain Transaction Analysis - Analyze blockchain transactions for suspicious patterns and risks
  • Wallet Risk Scanning - Scan wallet addresses for security threats and anomalies
  • Security State Summaries - Get comprehensive security posture assessments
  • Advanced Web Search - Powered by Exa MCP for high-quality, semantic search results

Technology Stack

  • AgentKit - Coinbase's framework for building AI agents
  • VeriSense - Security-focused agent platform
  • MCP (Model Context Protocol) - For tool integration and agent coordination
  • A2A Coordination - Automatic agent-to-agent communication for complex tasks
  • Exa Search - Advanced semantic web search capabilities

How It Works

WebWatcher uses intelligent agent coordination to analyze security threats. When you scan a website, multiple specialized agents work together:

  • UrlFeatureAgent extracts URL features and patterns
  • PhishingRedFlagAgent analyzes features for phishing indicators
  • Results are automatically coordinated via A2A protocol

Usage

Simply type your security questions or use the quick action buttons above the input field. Try commands like:

  • "scan website example.com" - Scan for phishing risks
  • "search CVE log4j" - Find vulnerability information
  • "analyze transaction 0x..." - Analyze blockchain transactions
